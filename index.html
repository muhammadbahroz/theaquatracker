    Product Description

    Our product is The Aqua Tracker which includes Human Machine Interface (HMI). The major aim is to save water, regulate, optimize and control water flow from remote locations via the internet. To achieve this, our design includes the installation of an intelligent and plug-and-play type product. This product is primarily composed of a microcontroller (IoT device) and different waterproof sensors.
    This product will interact with the internet and the cloud and continuously store readings from the sensors into a database. This database will make up a data set that can be used after a while to train the product to make decisions. Thus, making it an intelligent product. It also includes a generic volume fining functionality for each tank it is interfaced in. Other than that the user can access the product via the Human Machine Interface (HMI). Which is an Android based application with cloud access. The user can set tank levels, monitor tank levels on run time, display consumption records daily or monthly.
